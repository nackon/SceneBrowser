# SceneBrowser 仕様書 v0.1

## 1. 概要

SceneBrowser は、macOS 向けのローカル動画コレクション管理アプリケーションである。

主目的は、動画ファイルを「複数サムネイル」により視覚的に一覧・検索・管理することであり、単なる動画プレイヤーではなく「動画ブラウザ」として設計する。

WhiteBrowser のような「内容を視覚的に探す体験」を、macOS ネイティブに近い UX で実現する。

---

# 2. ターゲットユーザー

## 主対象

* 大量のローカル動画を保有しているユーザー
* 動画コレクション管理ユーザー
* 外付け HDD / NAS に動画を保存しているユーザー
* Finder の単一サムネイル表示に不満を持つユーザー

## 想定ユースケース

* 動画内容を視覚的に探したい
* シーンを見て目的動画を判別したい
* タグやレーティングで整理したい
* 大量ファイルを高速に閲覧したい

# 5. コアコンセプト

## 5.1 複数サムネイル表示

1動画につき複数サムネイルを生成する。

例:

* 4枚

動画内容を視覚的に把握できることを最優先とする。

# 6. MVP 機能

## 6.1 フォルダ登録

### 概要

監視対象フォルダを登録する。

### 要件

* 複数フォルダ登録
* 再帰検索
* 外付け HDD 対応
* NAS 対応

### 対応形式

* mp4
* mkv
* avi
* mov
* webm
* m4v

---

## 6.2 動画スキャン

### 概要

登録フォルダ内の動画を DB に登録する。

### 要件

* 差分スキャン
* 更新日時監視
* ハッシュ生成
* 重複検出

### メタ情報取得

ffprobe により以下を取得:

* 再生時間
* 解像度
* コーデック
* フレームレート
* ファイルサイズ

---

## 6.3 サムネイル生成

### 概要

動画から複数サムネイルを生成する。

### 初期仕様

* デフォルト16枚
* 等間隔抽出
* JPEG 形式
* 非同期生成

### ffmpeg 例

```bash
ffmpeg -i input.mp4 \
-vf "fps=1/60,scale=320:-1,tile=4x4" \
out.jpg
```

---

## 6.4 一覧表示

### 表示形式

* グリッド表示
* 可変サムネイルサイズ
* 無限スクロール

### 表示内容

* サムネイル

* 再生時間
* タグ
* レーティング

---

## 6.5 動画プレビュー

# 10. データベース設計（初期案）

## videos

| カラム        | 型        | 内容      |
| ---------- | -------- | ------- |
| id         | INTEGER  | PK      |
| path       | TEXT     | ファイルパス  |

| duration   | INTEGER  | 秒       |
| width      | INTEGER  | 横幅      |

| size       | INTEGER  | ファイルサイズ |

| updated_at | DATETIME | 更新日時    |

| カラム  | 型       | 内容  |
| ---- | ------- | --- |

| -------- | ------- | --------- |
